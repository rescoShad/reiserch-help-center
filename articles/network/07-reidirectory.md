**Purpose:** Search and browse all ReiSearch users to find potential partners, buyers, and collaborators.

**Content:**

The ReiDirectory is a searchable database of every registered ReiSearch user. Use it to find people by name, location, or expertise, and connect with professionals who match your needs.

### Accessing the Directory

Click "Directory" in the left sidebar or go to /rei-directory. The page shows a search bar at the top and a paginated results table below.

### Search and Filtering

Three ways to narrow results:

**Search by name** — Type a user's full or partial name in the search field. Results filter as you type.

**Search cities only** — Toggle to city-only search. Enter a city name to find users in that location. Useful when you are looking for partners in a specific market.

**Select Expertise** — Dropdown filter for expertise categories: Buyer, Acquisitions, Lead Generator, Wholesaler, and others. Select one to see only users with that expertise badge.

Combine filters for precise searches. For example: search for "Buyer" expertise in "Phoenix" to find active buyers in that market.

### Directory Table

Results display in a table with these columns:

| Column | What It Shows |
|--------|--------------|
| **Name** | User name with profile photo, location (city/state or email), and expertise badges as colored pills. |
| **Expertise** | Badge tags for each user's selected expertise areas. |
| **Phone Number** | Visible based on the user's phone visibility setting. May be hidden. |
| **Email Address** | Visible based on the user's email visibility setting. May be hidden. |
| **Actions** | Connect icon and View Profile icon for each user. |

### Connecting from the Directory

Click the Connect icon on any row to send a connection request. Click the profile icon to view the user's full profile before deciding.

### Pagination

Directory results are paginated:

- **Page size** — Choose 10, 25, 50, or 100 results per page from the dropdown.
- **Max results** — Directory returns up to [DIRECTORY_MAX_RESULTS] total results. If your search returns more, narrow your filters.
- **Page navigation** — Use numbered page buttons and Previous/Next links.

### Inviting from the Directory

The blue "Invite to ReiSearch" button at the top lets you invite people who are not yet on the platform. Enter their name and email. An invitation is sent with a sign-up link.

### Privacy in the Directory

What users see depends on the target's privacy settings. Phone numbers and emails marked Private do not appear. Users listed as Private in visibility settings may still appear in the Directory but without contact information.
