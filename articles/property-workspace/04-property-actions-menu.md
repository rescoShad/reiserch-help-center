## Help Content

**Title:** Property Actions Menu
**Format:** Help article
**Audience:** All
**Slug:** property-actions-menu
**Purpose:** Find every action available on a property detail page.

**Content:**

The property detail page is the control center for each property. Most actions are accessible from the buttons, tabs, and menus on this page.

### Primary Action Buttons

Located near the top of the property detail page:

**Run Comps** — Start an AI comparable market analysis. Pulls comps based on the property address and characteristics.

**Generate Presentation** — Create a deal presentation from the property data. Requires comps for full data panels.

**Share** — Share the property with specific users, your team, or generate a private link.

**Publish to Marketplace** — Make the property visible to all ReiSearch users in the Marketplace.

### Bottom Panel Tabs

The bottom panel or tab bar contains:

**Property Details** — Edit the property information: type, beds, baths, sq ft, price, year built, deal structure, status. Changes save automatically.

**Comps** — View comps attached to this property. Run new comps, review saved comps, or remove comps from the property.

**Presentations** — Create, view, and manage presentations linked to this property.

**Documents** — Upload, view, and download documents (contracts, reports, inspection files, photos).

**Threads** — Team discussion threads about this property. Reply, @mention, and attach files.

**Deal Tracker** — Track the property through pipeline stages. Move between New, Under Review, Offer Made, Under Contract, Closed, and Passed.

### Context Menu (Three-Dot Icon)

The three-dot icon usually opens additional actions:

- **Add to Folder** — Add the property to one or more custom folders.
- **Duplicate Property** — Create a copy of the property as a new draft. Useful for similar deals.
- **Export Data** — Download property data as CSV or PDF.
- **Archive** — Remove from active portfolio view but keep in your account.
- **Delete** — Permanently delete the property and all associated data. No undo.

### Keyboard Shortcuts

- **Ctrl+S / Cmd+S** — Save current changes.
- **Ctrl+P / Cmd+P** — Open presentation generator.

### Actions by Property Status

Some actions are only available at certain statuses:

| Action | Draft | Shared | Published | Offers |
|--------|-------|--------|-----------|--------|
| Edit details | Yes | Yes | Yes | Yes |
| Run comps | Yes | Yes | Yes | Yes |
| Generate presentation | Yes | Yes | Yes | Yes |
| Share | Yes | Yes | No* | No* |
| Publish | Yes | Yes | N/A | N/A |
| Unpublish | N/A | N/A | Yes | Yes |
| Move to Offers | No | Yes | Yes | N/A |
| Delete | Yes | No** | No** | No** |

*Already shared or published. Use Unpublish first.
**Archive instead of delete.
