## Help Content

**Title:** How to Browse the Marketplace
**Format:** Help article
**Audience:** New users
**Slug:** browse-marketplace
**Purpose:** Find and explore published properties on the ReiSearch Marketplace map and grid.

**Content:**

The Marketplace is where published properties from every ReiSearch user appear. It combines an interactive map with a property card grid so you can browse by location, see what is available, and drill into listings that interest you.

### Step 1: Open the Marketplace

Click "Marketplace" in the left sidebar or go to /home-dispo. The page loads with two panels side by side.

Left panel: Interactive map with property pin locations.
Right panel: Scrollable grid of property cards.

### Step 2: Navigate the Map

The map shows every published property with a colored pin. Each pin represents one listing.

- **Zoom and pan** — Use the mouse wheel or trackpad to zoom in and out. Click and drag to move around the map. As you zoom in, pins for nearby properties separate and become individually selectable.
- **Area-based search** — Zoom into a neighborhood or city and the grid updates to show only properties in that map area.

### Step 3: Browse Property Cards

The right panel displays property cards in a grid layout. Each card shows:

- **Property photo** with price overlay (e.g., $385,000)
- **Deal type badge** — Color-coded: Cash (red), Subject-To (teal), Seller Finance (green), Hybrid (yellow), Wrap (purple)
- **Address** with city and state
- **Beds, baths, and square footage**

Scroll down to load more properties. The grid updates as you move or zoom the map.

### Step 4: Select a Property

Click any property card or map pin to open the property detail page. From there you can:

- View full details, photos, and description
- Check market value data from Zillow, Redfin, and Realtor.com
- See the deal structure and terms
- Contact the publisher through the messaging system
- Run comps or generate a presentation

### Step 5: Use Filters to Narrow Results

The filter bar at the top of the page lets you narrow by location, deal type, price, and property characteristics. See "How to Use Marketplace Filters" for a full walkthrough.
