## Help Content

**Title:** How to Use Marketplace Filters
**Format:** Help article
**Audience:** New users
**Slug:** marketplace-filters
**Purpose:** Narrow or expand Marketplace results to find exactly the properties you are looking for.

**Content:**

The Marketplace filter bar sits at the top of the page and controls what appears on both the map and the property card grid. Each filter is a dropdown chip. You can use one filter or combine several to build precise searches.

### Filter Overview

The filter bar includes these options:

| Filter | What It Does |
|--------|-------------|
| Select City, Zip or State | Searches by location. Type a city, ZIP code, or state. |
| Deal Type | Filters by deal structure: Cash, Subject-To, Seller Finance, Hybrid, Wrap. |
| Price | Sets a minimum and maximum price range. |
| Year Built | Filters by construction year or year range. |
| Stories | Filters by number of stories. |
| Beds | Filters by minimum and maximum bedroom count. |
| Baths | Filters by minimum and maximum bathroom count. |
| Home Type | Filters by property type: Single Family, Multi-Family, Condo, Townhome, etc. |

### Using Filters

Click any filter chip to open its dropdown. Make your selection. The map and grid update immediately with matching results.

**To set a price range** — Enter a minimum value in the left field and a maximum in the right field. Leave one blank for an open-ended range (e.g., leave Minimum blank to find everything under $500,000).

**To select multiple deal types** — Open the Deal Type filter. Check each type you want to include. Filtering by Cash and Seller Finance shows properties in either category.

**To filter by a specific area** — Type in the location filter. You can add multiple locations. The system searches across all selected areas.

### How Filters Interact with the Map

The map and grid are connected. Apply a filter and both update to show only matching properties. If you zoom or pan the map, the grid updates to match the visible area. This means you can:

- Zoom into a neighborhood to see only properties there.
- Pan to a different part of the city to refresh the grid automatically.
- Combine map navigation with filter chips for precise results.

### Tips for Narrowing Results

If the Marketplace shows too many properties to browse:

- Start with location. Select a single city or ZIP code.
- Add one deal type at a time to see what is available in each category.
- Use the beds filter. Narrowing to 3+ bedrooms typically cuts the list significantly.
- Set a price ceiling to remove properties outside your budget.

### Tips for Widening Results

If a filter returns zero or very few properties:

- Remove the location filter and try a broader area (switch from a ZIP code to the full city).
- Check fewer deal types. Some deal types have limited inventory.
- Expand your price range by $50,000-$100,000 in each direction.
- Remove optional filters (Stories, Year Built) to see all available properties.

### Clearing Filters

Click "Clear All" or remove individual filter chips by clicking the X on each one. With no active filters, the Marketplace shows every published property within the current map view.
