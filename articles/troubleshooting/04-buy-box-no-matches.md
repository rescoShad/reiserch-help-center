## Help Content

**Title:** My Buy Box Has No Matches
**Format:** Troubleshooting article
**Audience:** All
**Slug:** buy-box-no-matches
**Purpose:** Understand why your buy box returns no results and how to broaden your criteria.

**Content:**

Buy boxes define the properties you want to buy. If yours is returning zero matches, the criteria may be too narrow, or no matching properties are currently available in the marketplace.

### Criteria Too Restrictive

The most common cause is a buy box with narrow criteria. Check each field:

- **Price range** — A tight spread (e.g., $100,000-$110,000) eliminates many properties. Try widening by $50,000 in each direction.
- **Property type** — Selecting only "Single Family" excludes all others. Try adding Multi-Family, Condo, or Townhome.
- **Location** — A single city or ZIP code may have no current inventory. Try expanding to the metro area or adjacent cities.
- **Beds and baths** — Requiring exact numbers shrinks the pool. Try "3+" instead of "3" or remove the filter entirely.
- **Deal type** — Selecting only "Cash" or only "SubTo" limits results. Try checking multiple deal types.

### No Properties Match in the Marketplace

It is possible that no published properties currently match your criteria. The Marketplace inventory changes as users publish and unpublish listings.

- Check back regularly. New properties are added daily.
- Widen your criteria temporarily to see what inventory exists, then narrow as matches appear.
- Connect with more users who publish properties matching your desired profile.

### Buy Box Is Not Active

Verify your buy box status. A draft buy box does not trigger matches.

- Go to your Profile or Settings and open your buy boxes.
- Confirm the buy box is marked active or published.
- If it is in draft, complete the required fields and save.

### Auto-Send Is Not Working

If you have auto-send enabled but are not receiving presentations:

- Auto-send is capped at 3 presentations per buyer per 7 days. If you have received 3 recently, the system waits before sending more.
- Verify the buy box is connected to a valid buyer profile with contact information.
- Check that the publisher has published properties that match your criteria.

### Data Refresh Needed

Marketplace listings and buy box matching refresh periodically. If you updated your buy box recently, allow a few minutes for the system to re-run matches.
