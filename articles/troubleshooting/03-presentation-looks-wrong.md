## Help Content

**Title:** My Presentation Looks Wrong
**Format:** Troubleshooting article
**Audience:** All
**Slug:** presentation-looks-wrong
**Purpose:** Fix formatting, layout, or data issues in your ReiSearch presentation.

**Content:**

Presentations pull data from the property profile and comp results. If your presentation looks incomplete, misaligned, or is missing content, the issue is usually in the source data.

### Missing or Incorrect Data

Presentations populate from the property detail page. If a field is empty or wrong in the presentation, update it on the property profile first.

- Open the property and check the Property Details panel, Deal Structure panel, and Market Value section.
- Correct any missing or inaccurate values.
- Return to Presentations and regenerate. The new version pulls the updated data.

### Wrong Images or Gallery Layout

The presentation image gallery pulls from the property's main photo and thumbnail strip.

- If the wrong image is showing, update the property gallery on the property detail page. Drag photos to reorder. The first photo becomes the cover image.
- If images appear stretched or cropped, use landscape-oriented photos. Portrait or low-resolution images may render poorly in presentation layouts.

### Presentation Looks Like a One-Pager

Free plan presentations generate as one-pagers with limited data and layout options. Starter and Core plans unlock standard multi-page presentations with full data panels. Scale plans include advanced templates and layout controls.

- If you expected a full presentation, check your current plan in Billing.
- Upgrade to generate presentations with complete data, multiple pages, and template options.

### Offer Accelerator Issues

Offer Accelerator creates rapid offer packages. If the generated output looks incomplete:

- Verify that the property has comps attached. The Offer Accelerator pulls comparison data from attached comp results.
- Check that deal structure fields (ARV, repair costs, wholesale contract amount) are filled in.
- Regenerate after fixing the source data.

### Layout or Alignment Problems

If text overlaps, images misalign, or sections break across pages:

- Try a different template (available on paid plans).
- Reduce the amount of content in Notes or custom text fields. Very long entries can break layout boundaries.
- Export as PDF and check again. Screen preview and PDF render may differ slightly.
