## Help Content

**Title:** How to Run Comps
**Format:** Help article
**Audience:** New users
**Slug:** how-to-run-comps
**Purpose:** Generate an AI-powered comparable market analysis for any property in under 60 seconds.

**Content:**

ReiSearch Comps gives you a full market analysis using AI. Enter an address, and the platform pulls comparable properties based on location, size, and recent sales.

### Step 1: Open the Comps Tool

From the Dashboard, click "Run Comps" or open any property and click "Run Comps" on the property detail page.

If you are starting from the Dashboard, type the property address in the input field. Start with the street number. Results populate as you type. Select the address from the dropdown.

[SCREENSHOT: Comps input field with partial address showing autocomplete dropdown]

### Step 2: Wait for AI Analysis

Comps run in the background. The AI processes the property details and searches for comparable sales. This usually takes a few seconds.

If you navigate away, check the Notifications bell when you return. Comp results appear there with a link to the results page.

### Step 3: Review the Comparable Table

When results are ready, the comps table shows 6-10 comparable properties. Each row includes:

- **Property address** — The comp property location.
- **Sale price** — Recent sale price.
- **Distance** — Distance from your subject property.
- **Key differences** — Differences in size, beds, baths, condition.
- **Comps score** — The AI's assessment of how comparable each property is.

[SCREENSHOT: Comps results table showing comparable properties with price, distance, and score columns]

### Step 4: Use the Comps Score

Each comparable property gets a score. Higher scores mean the property is more comparable to your subject property. Use the score to weigh which comps matter most in your analysis.

Properties with very low scores may have significant differences (different property type, very far away, or sold much earlier). Consider excluding them from your final analysis.

### Step 5: Adjust Filters

Use the filter bar to narrow results:

- **Sale date range** — Show only recent sales or include older data.
- **Distance radius** — Expand or shrink the search area.
- **Property type** — Limit to single family, condo, or multi-family only.
- **Square footage range** — Narrow by property size.

Results update instantly as you adjust filters.

[SCREENSHOT: Filter bar with distance radius slider and date range selector highlighted]

### Step 6: Export or Save

- **Export as PDF** — Download the comps report as a PDF document.
- **Export as CSV** — Download the comps data as a spreadsheet file.
- **Save to Property** — Attach the comp results to the property profile. They appear in the comps section of that property.
