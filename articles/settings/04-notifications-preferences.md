## Help Content

**Title:** Notification Preferences
**Format:** Help article
**Audience:** All
**Slug:** notifications-preferences
**Purpose:** Control which notifications you receive and how they are delivered.

**Content:**

ReiSearch sends notifications for property updates, network activity, messages, and system alerts. You control what you see and how you see it.

### Accessing Notification Settings

Go to Settings > Notifications. The page is organized by category. Each category has separate controls for email and in-app delivery.

### Notification Categories

**Property updates**
- Comp results ready.
- Property shared with you.
- Property status changes (on properties you are involved in).
- Price changes on watched properties.

**Network activity**
- New connection requests.
- Connection requests accepted.
- Buy box matches.
- AI matchmaking suggestions.

**Messages**
- New direct messages.
- Replies in property threads.
- @mentions.

**System notifications**
- Token balance warnings (low balance, zero balance).
- Plan changes or renewal reminders.
- Platform announcements.
- Account security alerts.

### Per-Category Controls

For each category, you can set:

**In-app notifications** — On or off. Controls appear in the Notifications bell and Dashboard panel.

**Email notifications** — Choose from:
- Instant — Every event sends an email immediately.
- Daily digest — All events from the day arrive in one email.
- Weekly digest — A weekly summary email.
- Off — No email notifications for this category.

### Quiet Hours

Set quiet hours to suppress notifications during specific times:

1. Go to Settings > Notifications.
2. Scroll to "Quiet Hours."
3. Set the start and end time (e.g., 10:00 PM to 7:00 AM).
4. Choose which categories are silenced during quiet hours.
5. Save.

During quiet hours, notifications are queued and delivered when quiet hours end. Urgent notifications (account security, password changes) may still come through.

### Critical Notifications

Some notifications bypass your quiet hours and frequency settings:

- Account security alerts (login from new device, password change).
- Billing notifications (payment failure, plan expiration).
- Platform-wide announcements (maintenance, outages).

These cannot be turned off.

### Checking Past Notifications

The Notifications bell shows recent notifications. Older notifications are accessible from the Notification History page, which is linked from the Notifications panel.

### Notification Count

The bell icon shows an unread count. The count displays up to 99+. Clear notifications by reading or dismissing them. Unread notifications are marked with a dot or bold text.
