## Help Content

**Title:** Promo and Discount Codes
**Format:** Help article
**Audience:** All
**Slug:** promo-discount-codes
**Purpose:** Apply promo codes, referral codes, and limited-time discounts to your account.

**Content:**

Promo codes give you access to discounted pricing, bonus tokens, or special offers on ReiSearch plans and bundles.

### Types of Codes

**Promo codes** — Limited-time offers distributed through email, social media, or partner channels. Common types:
- Percentage discount on a plan (e.g., 20% off first month).
- Fixed discount on a token bundle.
- Free bonus tokens with a bundle purchase.
- Free plan upgrade for a trial period.

**Referral codes** — Personal codes tied to your account. When someone signs up using your code, both you and the new user earn bonus tokens. Your referral code is available in Settings > Billing.

**Partner codes** — Codes from affiliated real estate programs or organizations. Example: SubTo students get 250 free presentations with the code "SubTo." Partner codes have specific eligibility requirements.

### How to Apply a Code

1. Go to Settings > Billing or /billing.
2. Click "Apply Promo Code" or "Add Code."
3. Enter the code exactly as provided (codes are case-sensitive).
4. Click "Apply."

The discount or bonus is applied immediately. Check the updated pricing or token balance to confirm.

### Code Rules

- One promo code per account at a time, unless stated otherwise.
- Referral codes and promo codes can be combined in some cases.
- Codes may have expiration dates. Expired codes cannot be applied.
- Codes may be limited to specific plan tiers or user types.

### Where to Find Codes

- **Email** — ReiSearch sends promotional offers to subscribed email addresses.
- **Notifications** — In-app notifications for active promotions.
- **Partner communications** — If you are part of a partner program, check their communications for exclusive codes.
- **Dashboard banners** — Active promotions are sometimes displayed as banners on the Dashboard.

### Checking Code Validity

If a code does not work:

- Check the expiration date.
- Confirm the code is entered exactly as provided (spelling and case).
- Verify the code applies to your plan or purchase type.
- Some codes are for new users only and cannot be applied to existing accounts.

If none of these apply, contact Support with the code and where you found it.

### Code History

Applied codes appear in your Billing history. You can see what was applied, when, and the discount or bonus you received.
